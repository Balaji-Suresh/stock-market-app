## User Flow - Viewing Stock Chart

```
Start
  │
  ├─→ Go to Trading Tab
  │
  ├─→ Select Stock (Dropdown or Watchlist)
  │
  ├─→ Click "Get Quote"
  │
  ├─→ Stock Data Loads
  │
  ├─→ Click on Stock Price Card
  │
  ├─→ Modal Opens with Chart
  │
  ├─→ View Chart (Default: 1D)
  │
  ├─→ Switch Timeframes (1D/3D/1W/1Y/5Y)
  │
  ├─→ View Statistics (High/Low)
  │
  ├─→ Read AI Recommendation
  │
  ├─→ Review Sector Analysis
  │
  ├─→ Make Decision
  │
  ├─→ Click Buy/Sell
  │
  └─→ Execute Trade
```

## User Flow - Exploring Sector Recommendations

```
Start
  │
  ├─→ Click "AI Insights" Tab
  │
  ├─→ Select Sector (Banking/IT/FMCG)
  │
  ├─→ View Sector Overview
  │   ├─ Rating
  │   ├─ Trend
  │   └─ Risk Level
  │
  ├─→ Review Top Stocks
  │   ├─ Price
  │   ├─ Change
  │   └─ Recommendation
  │
  ├─→ Read Analysis
  │   ├─ Positive Factors
  │   ├─ Risks
  │   └─ Opportunities
  │
  ├─→ Click Stock to Trade
  │
  ├─→ Redirected to Trading Tab
  │
  ├─→ Stock Auto-Selected
  │
  ├─→ Click "Get Quote"
  │
  ├─→ View Chart & Details
  │
  └─→ Execute Trade
```
